## Features

- Single-page view switching (no router) via local state
- Accessible landmarks, headings, and ARIA labels
- Simple chat interface with local state and simulated bot reply
- Vite + React 18 setup mirroring V1 dependency versions

## Scripts

```bash
npm install
npm run dev      # start Vite dev server
npm run build    # production build in dist/
npm run preview  # preview production build locally
```
